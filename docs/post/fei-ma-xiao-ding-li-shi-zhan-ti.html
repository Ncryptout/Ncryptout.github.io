代码很好读，把p和q单独作为两个明文加密，给出了用来加密他们的pq之间的关系式

hint1 = pow(2020 * p1 + q1, 202020, n1)
hint2 = pow(2021 * p1 + 212121, q1, n1)
hint3 = pow(2020 * p2 + 2021 * q2, 202020, n2)
hint4 = pow(2021 * p2 + 2020 * q2, 212121, n2)

第二部分思路是构造一个kq1的值 用他和n1取公约数，第三部分同理。
用费马小定理化简得
h2 ≡2021p1+212121(mod q1)
p1=(h2-212121)/2021 %q1
带入进h1的式子
h1≡2020^202020⋅p 1^202020 (modq1)
两边都%q1，直接相减就能得到kq1。
思路挺巧妙的 说出来轻松，做的时候博主已经卡俩小时实在没招了
